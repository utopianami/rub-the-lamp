
## 폰트 설치

```bash
brew install --cask font-d2coding-nerd-font
```

## 설정 경로

iTerm2 → Settings (⌘,) → Profiles → Text

|항목|값|

|Draw bold text in bold font|✅|
|Use built-in Powerline glyphs|✅|
